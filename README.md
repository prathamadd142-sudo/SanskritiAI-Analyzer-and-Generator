# SanskritiAI-Analyzer-and-Generator
is a Python-based yoga pose analytics tool that decodes Sanskrit pose names, evaluates their biomechanical impact (flexibility, strength, balance, and intensity), and provides intelligent cool-down recommendations with visual pose comparisons.
The project combines Sanskrit linguistic parsing, data analysis, and visualization to create a Computational Linguistics & Data Analytics yoga analytics system.

Features:
🧠 Pose Difficulty Analyzer
Classifies yoga poses into Beginner, Intermediate, or Advanced based on intensity.
🔍 Sanskrit Pose Parser
Breaks down Sanskrit pose names into their root meanings.
Utilizes a mapping algorithm to decompose compound Sanskrit words into primary roots (Dhatus) and modifiers.
🧘 AI Cool-Down Recommendation
Suggests relaxing poses after high-intensity poses.
📊 Pose Comparison Visualization
Compare two poses using biomechanical radar chart analytics.
🤖 AI Pose Generator
Generates new Sanskrit-style yoga pose names using root combinations.

🛠️ Built With:
Python
Pandas
NumPy
Matplotlib

💡 Project Goal:
The goal of Sanskriti-AI is to demonstrate how data science, Sanskrit linguistic parsing, and simple AI logic can be used to analyze and visualize yoga poses.
